# Anthra Systems Website

Astro-based website for Anthra Systems, focused on custom electronics, embedded systems, industrial IoT, automation, smart devices, edge AI, connected vehicle systems, mechatronics, water and agriculture systems, and applied R&D.

## Development

```bash
npm install
npm run dev
npm run build
```

## Project Notes

- Main pages live in `src/pages`.
- Shared content and site metadata live in `src/data_files`.
- Solution listing and solution detail data live in `src/data_files/solutions.ts`.
- Visual assets live in `src/images`.
- The site should keep the Anthra visual direction: premium dark graphite, practical engineering imagery, controlled orange accents, and client-friendly technical clarity.

## SEO Direction

The website metadata is tuned for Anthra's real work:

- custom electronics solutions
- embedded systems development
- industrial IoT dashboards and gateways
- industrial automation and control
- smart device development
- edge AI and computer vision
- connected vehicle IoT
- mechatronics and prototypes
- water, agriculture, mobility, and field monitoring systems
- applied R&D and productization
